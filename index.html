SWITCHjs Menu:

Conversion/decompression tools for some Nintendo Switch file types

Note: These tools were each made for some of the Nintendo Switch games MK8D, SMG1 (in SM3DAS), SM3DW+BF, SMBW, SMM2, and SMO. They may or may not work with files from other games or consoles.

Explorer Explore the contents of files (implemented types listed below, and also zip files)

YAZ0 YAZ0, U8 (.szs, .arc) to BIN (.bin) - Used for compression.

SARC SARC (.sarc, .pack, .aras, .baatarc, .lyarc) to ZIP (.zip) - An archive.

RARC RARC (.rarc) to ZIP (.zip) - Another type of archive.

BYML BYML (.byml, .bgyml, .byaml) to JSON (.json) - Used for data/info storage.

BCSV BCSV (.bcsv, .banmt, .bcam, .pa, .tbl) to CSV (.csv) - Stores a table of data.

MSBT MSBT (.msbt) to JSON (.json) - Stores text.

BMG BMG (.bmg) to CSV (.csv) - Also stores text.

BARSLIST BARSLIST (.barslist) to JSON (.json) - Stores lists of audio file names.

BFSAR BFSAR (.bfsar) to ZIP (.zip) - An archive of audio-related files and data.

BFGRP BFGRP (.bfgrp) to ZIP (.zip) - An archive of audio files.

BFWAR BFWAR (.bfwar) to ZIP (.zip) - An archive of wave audio files.

BFWSD BFWSD (.bfwsd) to JSON (.json) - Stores information about and references to wave audio files.
