Conversion/decompression tools for some Nintendo Switch file types

Note: These tools were each made for some of the Nintendo Switch games MK8D, SMG1 (in SM3DAS), SM3DW+BF, SMBW, SMM2, and SMO. They may or may not work with files from other games or consoles.

Explorer Explore the contents of files (implemented types listed below, and also zip files)

YAZ0 YAZ0, U8 (.szs, .arc, .sbfarc, .sesetlist, .sbmaptex, .sbmapopen, .sbitemico, .sbstftex, .sbreviewtex, .sstats) to BIN (.bin) - Used often for compression.

SARC SARC (.sarc, .pack, .aras, .baatarc, .lyarc) to ZIP (.zip) - A common archive of other files.

RARC RARC (.rarc) to ZIP (.zip) - Another type of file archive.

BYML BYML (.byml, .bgyml, .byaml) to JSON (.json) - Often used for data or info storage.

BCSV BCSV (.bcsv, .banmt, .bcam, .pa, .tbl) to CSV (.csv) - Stores a table of data.

MSBT MSBT (.msbt) to JSON (.json) - A common format that stores text.

BMG BMG (.bmg) to CSV (.csv) - Occasionally used to store text.
